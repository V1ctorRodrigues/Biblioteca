Bem vindo ao meu projeto inicial de uma simples biblioteca online com o intuito de trabalhar com as
boas práticas de POO.

Nesse projeto haverá as entidades:
Usuário: Nome, ID, empréstimos, atrasos.
Material: Tipo, nome, ID, disponibilidade, prazo.
Emprestimos: Recebe Usuário e Meterial.
Lista: Recebe todos os meteriais e exibe o status de cada.

O sistema fará o cadastro de usuários e materiais, vai listar esse material com a disponibilidade de cada
um e terá um sistema de empréstimos e devoluções.

Regras:
Não emprestar se estiver indisponível;
Não criar dois IDs iguais;
Não cadastrar vários usuários ou materiais idênticos;
Registrar empréstimo e devolução;
Mostrar atraso, se houver;
